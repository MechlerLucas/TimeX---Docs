# Requisitos Funcionais – TimeX

### RF01 – Criar Tarefa
O sistema deve permitir o cadastro de novas tarefas contendo: título, descrição, categoria (cor), data, hora e status.

### RF02 – Editar Tarefa
O sistema deve permitir a edição dos dados de uma tarefa existente.

### RF03 – Excluir Tarefa
O usuário pode excluir uma tarefa permanentemente.

### RF04 – Listar Tarefas
O sistema deve exibir todas as tarefas cadastradas, ordenadas por data/hora.

### RF05 – Filtrar Tarefas por Categoria
O usuário deve poder filtrar tarefas por categorias representadas por cores (ex.: Azul, Verde, Vermelho, Amarelo, Geral).

### RF06 – Visualizar Detalhes da Tarefa
O app deve exibir uma tela com as informações completas da tarefa ao ser selecionada.

### RF07 – Marcar/Desmarcar Tarefa como Concluída
O sistema deve permitir alterar o status de conclusão da tarefa.

### RF08 – Persistência Local
Todas as tarefas devem ser armazenadas localmente (JSON, filesystem ou banco local).

### RF09 – Criar e Gerenciar Categorias (Opcional)
O usuário pode criar, editar e excluir categorias personalizadas.

### RF10 – Exibir Data Atual na Tela Inicial
A tela inicial deve sempre mostrar a data atual.
